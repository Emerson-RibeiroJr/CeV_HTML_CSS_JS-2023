Tags em HTML

Em HTML5, podemos inserir os elementos da lista a seguir. Passe o mouse sobre os termos para descobrir qual tag deverá utilizar em cada caso.

Lembrando que essa função é feita utilizando a tag <abbr>

  • Parágrafo
  • Títulos
  • Imagens
  • Abreviações
  • Códigos-fonte
  • Listas Numeradas
  • Listas com Demarcadores (não numeradas)
