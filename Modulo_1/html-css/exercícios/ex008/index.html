Negrito / Destaque

Segue um exemplo de um termo em negrito nesta frase, utilizando a tag <b> porém essa tag não é semântica, apenas forma.

Agora neste caso, utilizaremos corretamente uma tag semântica para deixar em NEGRITO utilizando a tag <strong>. Dando Sentido e forma.

Itálico / Ênfase

Nesta frase, seguiremos com um exemplo de Itálico não semântico, ou seja, apenas forma, utilizando a tag <i>.

E agora utilizando da forma correta com semântica, dando sentido e forma, temos o Itálico, utilizando a tag <em>
